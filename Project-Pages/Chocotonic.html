Chocotonic Wrappers and Website

For this project I was tasked to create a hypothetical chocolate company, the logo, and wrapper designs. Chocotonic, the hypothetical company created, is a chocolate company with sustainability at its core, using proceeds from sales to donate to rainforest sustainability. I later used this same project as inspiration for a website project, creating a site that allows consumers to both buy directly from them as well as learn about their goals.
